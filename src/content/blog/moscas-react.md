Este proyecto es el prototipo de mi tesis. Es una aplicación web completa donde los usuarios pueden cargar imágenes de moscas de la fruta y el sistema clasifica automáticamente la especie.

### 🚀 Tecnologías Core

- **Frontend:** React (Componentes, Hooks, estado).
- **Backend:** Python.
- **Inteligencia Artificial:** Modelo de visión computacional para clasificación.

### 💻 ¿Qué problema resuelve?

Permite automatizar y acelerar la identificación de plagas agrícolas (Anastrepha fraterculus y Ceratitis capitata) a través de una interfaz web rápida e intuitiva.
